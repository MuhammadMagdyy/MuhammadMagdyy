### 📄 Stanza Studio — *Intelligent PDF Knowledge Engine*

> **"Turn any PDF library into a conversational knowledge base."**

**Stanza Studio** is a production-grade **RAG (Retrieval-Augmented Generation)** application that ingests PDF documents, chunks and embeds them into a **Qdrant vector database**, and serves natural-language queries through a **FastAPI backend** — delivering precise, source-cited answers at scale.

| Layer | Technology |
|:---|:---|
| 🧠 **AI Pipeline** | LangChain · OpenAI Embeddings · RAG Architecture |
| ⚙️ **Backend** | Python · FastAPI · Async Processing |
| 🗄️ **Vector Store** | Qdrant |
| 📦 **Infra** | Docker · REST API |

**Highlights:**
- 📥 Automated PDF ingestion with intelligent chunking strategies
- 🔍 Semantic similarity search across thousands of document pages
- 💬 LLM-powered Q&A with grounded, source-linked responses
- ⚡ Async FastAPI endpoints built for high-throughput querying

[![Repo](https://img.shields.io/badge/View_Repo-6E40C9?style=for-the-badge&logo=github&logoColor=white)](https://github.com/MuhammadMagdyy/stanza-studio)

### 🎓 Athar Studio — *Student Directory & Analytics Dashboard*

> **"A modern, data-rich student management experience."**

**Athar Studio** is a **React + TypeScript dashboard** for managing and visualizing student records. It delivers a polished, responsive UI with real-time filtering, search, and role-based views — bridging the gap between raw data and actionable insights for educators.

| Layer | Technology |
|:---|:---|
| 🎨 **Frontend** | React · TypeScript · Tailwind CSS |
| 📡 **Data Layer** | REST API Integration · React Query |
| 🖥️ **UI/UX** | Responsive Design · Component Architecture |

**Highlights:**
- 🔎 Real-time search & multi-criteria filtering across student records
- 📊 Visual analytics panels with enrollment and performance summaries
- 🔐 Role-based access views for administrators and instructors
- 📱 Fully responsive across desktop and mobile viewports

[![Repo](https://img.shields.io/badge/View_Repo-6E40C9?style=for-the-badge&logo=github&logoColor=white)](https://github.com/MuhammadMagdyy/athar-studio)
